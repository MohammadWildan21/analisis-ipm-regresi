# Analisis Korelasi dan Regresi Linier Berganda terhadap Indeks Pembangunan Manusia di Indonesia

## Deskripsi Proyek

Repositori ini berisi proyek analisis data yang bertujuan untuk menganalisis faktor-faktor yang berkaitan dengan **Indeks Pembangunan Manusia (IPM)** di Indonesia menggunakan pendekatan **Analisis Korelasi Pearson** dan **Regresi Linier Berganda**.

Analisis dilakukan menggunakan data **38 provinsi di Indonesia** yang bersumber dari **Badan Pusat Statistik (BPS) RI**.

---

## Tujuan

Proyek ini bertujuan untuk:

- Menganalisis hubungan antara variabel independen dengan IPM.
- Menyusun model regresi linier berganda.
- Mengevaluasi signifikansi model menggunakan uji statistik.
- Melakukan prediksi nilai IPM berdasarkan variabel independen.

---

## Variabel

### Variabel Dependen (Y)

- Indeks Pembangunan Manusia (IPM)

### Variabel Independen (X)

- Angka Harapan Hidup (AHH)
- Produk Domestik Regional Bruto (PDRB)
- Tingkat Kemiskinan
- Rata-rata Lama Sekolah (RLS)
- Tingkat Pengangguran Terbuka (TPT)
